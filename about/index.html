About

Me

I'm Jordon, a 20-year-old Software Engineer who is currently attending Rose-Hulman Institute of Technology. I love learning new things and exploring new technologies. Currently I'm trying to learn about Go in what spare time I have, and I'm trying to find the perfect project use it on. I've worked for BP3 in Austin, Texas, where I developed an internal tool that allowed the consultants to spin up and manage EC2 instances with only a few clicks and, most importantly, WITHOUT having to bother an admin. None of what I say here is affiliated with them or with any other current or future employer; this is an entirely personal blog.

This Blog

For those curious, this blog is powered by Jekyll, a simple, blog-aware static site generator. I'm hosting the site on Github Pages, which is absolutely free. Though I would have loved to create this site using a more advanced framework, such as the Revel Framework for Go, the allure of saving money drew me in. I'm eternally grateful to Github for providing this service to developers.

Additionally, I'm using the wonderful icons from fontawesome for my navbar icons, and Twitter Bootstrap 3 wip for all my CSS styling needs. Note that, since Bootstrap 3 has not had a full release yet, there will necessarily be some graphical hiccups here and there. In particular, the scaffolding is not all there at time of writing.
